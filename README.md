## 🛠 Wanna run on your machine?

1. Clone the project

   ```sh
   git clone https://github.com/ironsoul0/ironsoul.ninja.git
   ```

1. Install the dependencies

   ```sh
   cd ironsoul.ninja
   yarn
   ```

3. Start the development server

   ```sh
   yarn dev
   ```
